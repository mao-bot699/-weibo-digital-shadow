## 给B看的：scrapers.py 需要输出什么格式

### search_weibo_user(keyword) 返回格式

```python
# 返回：用户列表，每个用户是字典
[
    {
        "uid": 123456,           # 用户ID（数字）
        "screen_name": "昵称",   # 用户昵称
        "description": "简介",   # 用户简介
        "followers_count": 500,  # 粉丝数
        "follow_count": 200,     # 关注数
        "statuses_count": 1000,  # 微博数
    },
    ...
]
```

### get_weibo_user_info(uid) 返回格式

```python
# 返回：用户信息字典
{
    "screen_name": "昵称",
    "description": "简介",
    "followers_count": 500,
    "follow_count": 200,
    "statuses_count": 1000,
    "verified": False,           # 是否认证（布尔值）
    "verified_reason": "",       # 认证原因（字符串）
    "gender": "男",              # 性别
    "city": "深圳",              # 城市
}
```

### get_weibo_user_posts(uid) 返回格式

```python
# 返回：微博列表，每个微博是字典
[
    {
        "clean_text": "清洗后的纯文本内容",  # 必须是纯文本，不能有HTML
        "created_at": "2024-01-01 12:00",    # 发布时间
        "source": "iPhone 16 Pro",           # 发布来源（手机型号）
        "region_name": "广东深圳",           # 发布地区
        "reposts_count": 2,                  # 转发数
        "comments_count": 15,                # 评论数
        "attitudes_count": 50,               # 点赞数
    },
    ...
]
```

### 重要提醒

1. **clean_text 必须是纯文本**，不能有HTML标签、@用户、话题标签
2. **字段名必须和上面一模一样**，否则agent.py会报错
3. **如果某个字段拿不到，用空字符串""或0填充**，不要返回None

## 给前端看的：怎么调用 agent.py

### 调用方式

```python
from agent import analyze_weibo, continue_analysis

# 第一步：分析微博数据
report = analyze_weibo(user_info, posts)

# 第二步：追问分析（可选）
updated_report = continue_analysis(previous_report, new_info)
```

### 输入参数

**analyze_weibo() 需要两个参数：**

```python
# 参数1：user_info 字典
user_info = {
    "screen_name": "昵称",
    "description": "简介",
    "followers_count": 500,
    "follow_count": 200,
    "statuses_count": 1000,
    "verified": False,
    "verified_reason": "",
    "gender": "男",
    "city": "深圳",
}

# 参数2：posts 列表
posts = [
    {
        "clean_text": "微博内容",
        "created_at": "发布时间",
        "source": "发布来源",
        "region_name": "发布地区",
        "reposts_count": 0,
        "comments_count": 0,
        "attitudes_count": 0,
    },
    ...
]
```

**continue_analysis() 需要两个参数：**

```python
# 参数1：之前的分析报告（字符串）
previous_report = "之前的报告内容..."

# 参数2：用户补充的新信息（字符串）
new_info = "用户补充的内容..."
```

### 返回值

```python
# 两个函数都返回字符串（分析报告）
report = analyze_weibo(user_info, posts)
# report 是字符串，直接显示即可

updated_report = continue_analysis(previous_report, new_info)
# updated_report 是字符串，直接显示即可
```

### 错误处理

```python
# 如果出错，返回的字符串会以"分析失败："开头
report = analyze_weibo(user_info, posts)

if report.startswith("分析失败"):
    # 显示错误信息
    print(report)
else:
    # 正常显示报告
    print(report)
```

### 完整调用示例

```python
from scrapers import search_weibo_user, get_weibo_user_info, get_weibo_user_posts
from agent import analyze_weibo, continue_analysis

# 1. 搜索用户
users = search_weibo_user("昵称")
user = users[0]
uid = user["uid"]

# 2. 获取用户信息
user_info = get_weibo_user_info(uid)

# 3. 获取用户微博
posts = get_weibo_user_posts(uid)

# 4. 分析
report = analyze_weibo(user_info, posts)

# 5. 显示
print(report)

# 6. 追问（可选）
new_info = "用户补充的信息..."
updated_report = continue_analysis(report, new_info)
print(updated_report)
```
